## 📌 Описание проекта

React + TypeScript приложение, реализующее каталог фильмов с возможностью:
- отображения списка фильмов (плитка или список);
- добавления/удаления фильмов в избранное;
- фильтрации только избранных фильмов;
- поиска по названию (без учёта регистра);
- переключения режима отображения (плитка / список).

Все данные о фильмах захардкожены в коде (`src/data/movies.mock.ts`).

---

## 🧩 Логика реализации

### Основные компоненты

#### `App.tsx`
Главный компонент:
- хранит состояния:
    - `movies` — массив фильмов;
    - `filter` — `'all' | 'fav'`;
    - `view` — `'grid' | 'list'`;
    - `query` — строка поиска;
- производит фильтрацию и поиск через `useMemo`;
- передаёт отфильтрованные фильмы в `MovieList`;
- управляет избранным (`toggleFavorite`).

#### `Toolbar.tsx`
Панель инструментов:
- кнопки фильтрации (`Все`, `Только избранные`);
- поле поиска (`input` с `useRef`);
- переключатель вида (`Плитка` / `Список`);
- при кликах вызывает соответствующие коллбеки.

#### `MovieList.tsx`
- условно рендерит список фильмов;
- использует `MovieCard` для каждой карточки;
- если фильмов нет — выводит текст «Фильмов нет».

#### `MovieCard.tsx`
- показывает постер, название, год;
- кнопка ⭐ для добавления/удаления из избранного;
- оформление различается для режима `grid` и `list`.

## 🧠 Алгоритмы

**Фильтрация:**
```ts
let base = movies;
if (filter === 'fav') base = base.filter(m => m.isFavorite);
if (query.trim()) base = base.filter(m => m.title.toLowerCase().includes(query.toLowerCase()));
```
Избранное:
```ts
setMovies(prev => prev.map(m => (m.id === id ? { ...m, isFavorite: !m.isFavorite } : m)));
```
Переключение вида:
```ts
view === 'grid' ? карточки в сетке : вертикальный список
```
