## 🚀 Быстрый старт

Каждый проект находится в отдельной папке и имеет собственную конфигурацию.

### Установка и запуск любого проекта:

```bash
# Перейдите в папку проекта
cd HW_1/movie-catalog   # или cd HW_3

# Установите зависимости
npm install

# Запустите в режиме разработки
npm run dev
```

## 📖 Описание проектов

### HW_1 — Каталог фильмов

React + TypeScript приложение с каталогом фильмов.

**Возможности:**
- Отображение списка фильмов (плитка или список)
- Добавление/удаление фильмов в избранное
- Фильтрация только избранных фильмов
- Поиск по названию (без учёта регистра)
- Переключение режима отображения (плитка / список)

**Подробнее:** [HW_1/movie-catalog/README.md](./HW_1/movie-catalog/README.md)

---

### HW_3 — E-commerce Admin Panel

SPA административная панель для e-commerce системы.

**Возможности:**
- Аутентификация и авторизация (DummyJSON API)
- Защищённые маршруты
- Каталог продуктов с поиском и пагинацией
- Детальная страница продукта
- Профиль пользователя
- Настройки приложения (язык, тема, размер страницы)
- Интернационализация (русский / английский)
- Светлая / тёмная тема

**Архитектура:** Feature Sliced Design (FSD)

**Подробнее:** [HW_3/README.md](./HW_3/README.md)
